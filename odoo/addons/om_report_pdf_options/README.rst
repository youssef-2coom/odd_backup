==============
Report Options
==============

shows a modal window with options for printing, downloading or opening pdf reports

We just taken the code from this module: https://apps.odoo.com/apps/modules/14.0/report_pdf_options/
and made some adjustments to make it work for excel reports too.

Happy to merge with original module, if the author can reach us out.
