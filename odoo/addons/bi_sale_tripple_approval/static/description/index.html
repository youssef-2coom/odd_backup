Sales Order/Quotation Tripple Validation-Approval Workflow Odoo Apps

This Odoo apps helps you to add three-step approval process for Sales order while validation of Quotation in Odoo.This Odoo module add three setup validation process and triple approval validation process for Quotation-Sales order, only specific users like finance manager, Director have access to this kind of multi-step validation of Sales Order. Only user with "Finance Manager (Accounting)" groups can confirm sales from 'Second Approval' state and user with "Director(Accounting)" groups can Sales from 'Third Approval'. This Odoo app helps you to triple approval process for Sales order while validation of Quotation order.

Different Access Rights

for first and second order approval system has different roles like director and finance manager.

Triple Approval For Order
